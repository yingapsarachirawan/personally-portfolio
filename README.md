# Reachny Creative Portfolio

A responsive one-page portfolio built with React, Vite, Tailwind CSS 4, and Lucide icons.

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Edit the content

Most personal content is stored in:

```text
src/data/portfolio.js
```

Update the TikTok and Instagram links there if the current handle is different.

## Main assets

```text
public/assets/reachny-portrait.png
public/Reachny-Ying-An-CV.pdf
```

## Deploy

The generated `dist` folder can be deployed to Netlify, Vercel, Cloudflare Pages, or any static host.

Recommended build command: `npm run build`

Recommended publish directory: `dist`
